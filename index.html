How it Works: Reinforcement Learning

▼

What is RL?

Reinforcement Learning is an area of machine learning where an agent learns to make decisions by performing actions in an environment to maximize cumulative rewards.

The Q-Learning Algorithm

Q-Learning is a model-free RL algorithm that learns the value of an action in a particular state. It uses a "Q-Table" to store these values.

The Update Rule

Q(s,a) ← Q(s,a) + α [ r + γ max Q(s',a') − Q(s,a) ]

Exploration vs Exploitation

The agent uses an ε-greedy strategy: it mostly takes the best action (exploitation), but occasionally takes a random one (exploration) to discover new paths.
